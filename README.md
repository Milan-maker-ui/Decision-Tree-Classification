# Decision-Tree-Classification
A Machine Learning project demonstrating Decision Tree Classification using Python and Scikit-learn.

## Project Overview

This project implements a Decision Tree Classifier to:
- Load and preprocess data
- Train a Decision Tree model
- Predict class labels
- Evaluate model performance
- Visualize the Decision Tree

The project is built using:
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
